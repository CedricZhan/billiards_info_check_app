# Billiards Info Check App

A simple Flask web app to check snooker competition information.

## Features

* View snooker competitions
* Simple web interface

## How to run

1. Clone this repository:

```bash
git clone https://github.com/your-username/billiards_info_check_app.git
```

2. Navigate to the project folder:

```bash
cd billiards_info_check_app
```

3. Open a terminal in this folder and run:

```bash
python app.py
```

4. Then open your browser and visit:

```
http://127.0.0.1:5000/
```
